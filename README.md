Maya Water Tracker App

Aplicativo desenvolvido para registrar e acompanhar o consumo diário de água.
O design original está disponível em:
https://www.figma.com/design/P3wJasJoNAFcecdJBi8If4/Maya-Water-Tracker-App

Descrição

O Maya Water Tracker App permite que o usuário registre sua ingestão de água ao longo do dia e visualize seu progresso de forma simples e intuitiva.
O objetivo é incentivar uma rotina de hidratação saudável.

Instalação e Execução
1. Instalar dependências
npm install

2. Executar o projeto
npm run dev
